Уязвимости:
1. Path traversal в имени пользователя, можно например создать пользователя с именем ../db.db
2. SQL инъекция в DOWNLOAD
3. Слабая генерация хэша, можно подобрать пароль зная логин
4. Не thread-safe работа с бд. Если клиентов много, то случаются гонки, но если в тренировке только 1 чекер и 1 противник и если противник не будет ддосить, то норм)
